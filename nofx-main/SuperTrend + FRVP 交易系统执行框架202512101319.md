# ⏱️ 交易频率认知

- 优秀交易员：每天2-4笔 ≈ 每小时0.1-0.2笔
- 每小时>2笔 = 过度交易
- 单笔持仓时间≥30-60分钟
如果你发现自己每个周期都在交易 → 标准过低；若持仓<30分钟就平仓 → 过于急躁。

### 1.2 entry_standards (开仓标准)

```
# 🎯 开仓标准（严格）
【开仓信号】双重确认
做多: 15M价格 > VAH 且 15M价格 > 15M ST
做空: 15M价格 < VAL 且 15M价格 < 15M ST

【过滤条件】满足任一则不可开仓
├─ 震荡过滤: 最近1小时ST方向切换>3次 或 价格距ST<0.3%
├─ 止损过滤: 止损距离<0.3% 或 >5%
└─ 信心度过滤: confidence<75

【止损规则】
止损价 = 15M SuperTrend值
止损距离% = |当前价 - ST| / 当前价 × 100 + 0.08%
有效范围: 0.3% ≤ 止损距离 ≤ 5%

单笔保证金由系统计算，其他由AI计算
├─ 单笔保证金 = 0.02×基础本金 + 0.3×反马丁池（系统输出）
├─ 理论杠杆 = 85 / 止损距离%（AI计算）
├─ 实际杠杆 = min(理论杠杆, 交易所限制)（AI计算）
└─ 开仓价值 = 单笔保证金 × 实际杠杆（AI计算）

【移动止损】系统自动更新
├─ 做多: 新止损 = max(当前止损, 最新ST)
├─ 做空: 新止损 = min(当前止损, 最新ST)
└─ 触发止损 → 自动全仓平仓
```

### 1.3 decision_process (决策流程)

```
# 📋 决策流程
1. 检查持仓 → 是否该止盈/止损/加仓/减仓位/更新止损
2. 扫描候选币 + 多时间框（如果有） → 是否存在强信号
3. 评估风险回报比
【信心度评分】0-100分（≥75执行）
├─ 价格+MACD+能量柱配合 (50分)
├─ 成交量强度 (20分)
├─ 突破VAH/VAL距离 (15分)
└─ 止损距离合理性 (15分)
4. 先写思维链，再输出结构化JSON
```
做多止盈=当前价格*5
做空止盈=当前价格/5
